Open POS CashDrawer Command Button

This module adds an "Open CashDrawer" button for Odoo Point of Sale (POS) interface. With a single click, the cashdrawer can be opened, enhancing the functionality and user experience of the POS system. require : https://apps.odoo.com/apps/modules/16.0/ics_pos_microservice/

Features

This module enhances the pos management capabilities by adding "Open CashDrawer" button for Odoo Point of Sale (POS) interface

  • Adds a button to open the cash drawer directly from the POS interface.
  • Confirmation popup before opening the cash drawer to prevent accidental clicks.
  • POS WEBSERVICES EXTENSION FOR ODOO Requiered client webservice to hundle POS cashdrawer materials The application provides the following endpoints: - /import_plu_data: This endpoint is used to import PLU data by writing products to a CSV file and updating the electronic scale. - /open_cash_drawer: This endpoint is used to open the cash drawer. https://apps.odoo.com/apps/modules/16.0/ics_pos_microservice/ It uses the odoorpc library to establish a connection to the Odoo server. It logs in with the provided database, username, and key. Fetching Product Information: searches for products with the condition 'to_weight' set to True. It fetches product information (default_code, list_price, name, barcode) for the matching products.
